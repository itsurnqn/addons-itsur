Sale Price Change Restriction

In case user wants to restrict the sale price update to some of the user, this module could help them. After installing this module, user need to configure the rights to be provided to other users for changing the unit price of product. Here are the features of this module:

  • Easily grant user rights to change the unit price on Sales order.
  • Users outside the group will not be able change the Price of product to less than the Unit Price in Sale Order.
  • Only if the price list is selected for the Sale Order, the restriction will be enabled.
  • Also if a person not having this right tries to change the unit price of product of the price list present on sale order than an warning is raised.

Prerequisites before Installation

No prerequisites needed.

Configurations

User needs to Configure the Pricelist rights from the settings. The module will work only when the Price list is selected on the Sale Order and if then specific user with rights can restrict the price.

After Installation of the module the user provide rights to the user from Settings> Manage the Price Change.

Just check 'Manage Sales Price Change' is enabled inside User> Other Extra Rights settings

Note: Odoo developer mode is required.
